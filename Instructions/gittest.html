My Git Steps

Hello World its Matt!

  1. Clone Git "URL" in Git hub, into the DIR you want to use. Don't forget your README.txt
  2. You don't need to use "Git init" because your cloning your repository.
  3. Next step use "Git Status" to see what files you need to add. the files should be in "red".
  4. Next use the command "Git Add" to stage the files, you can use "Git Status to check, the files should now be green.
  5. Once the files are staged use "Git commit -m "updated ...". say what you changed in the quotations.
  6. the final step is to use "Git push origin master" this will upload your project to Github.

Be sure to check that your work is uploaded correctly to Github by checking the website. You should be able to see when you updated the repository. Good luck Matthew!
